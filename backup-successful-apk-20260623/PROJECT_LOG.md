# 项目过程记录

## 对话关键节点

1. 原项目是 Egern / iOS 小组件，不能直接在安卓上使用。
2. 先做过 PWA、多文件网页、单文件网页方案。
3. 因用户不能改服务器且遇到 CORS，转为同站脚本、书签脚本、油猴脚本。
4. 用户确认油猴脚本可在桌面 Edge 使用。
5. 用户进一步提出核心目标：安卓手机可安装、支持多个站点、最好可桌面使用。
6. 项目转向原生 Android App。
7. 已完成 Android App、多站点查询、桌面 Widget。
8. 已接入 GitHub Actions 自动打包 APK。
9. 用户已成功下载并安装 APK 到手机，可正常查询。

## 当前可回退版本价值

这是已经验证“可安装、可打开、可查询、可出包”的稳定版本。
后续所有界面美化、字段扩展、Widget 增强，都应以这个版本作为回退点。

## 下一步优化方向

- 查询结果页视觉优化
- 展示更多可用字段与衍生指标
- Widget 样式提升
- 正式签名和 Release APK
